# IT-Библиотека

Образовательное веб-приложение для изучения IT-терминов, технологий и специальностей.

## Описание

IT-Библиотека помогает начинающим разработчикам и всем интересующимся IT разобраться в технических понятиях через:
- Простые аналогии из жизни
- Примеры кода на разных языках
- Интерактивные схемы процессов
- Мини-квизы для проверки знаний
- Описания IT-специальностей и зарплат
- Архитектурные схемы реальных проектов

## Технологии

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- TanStack Query
- React Router
- Mermaid (диаграммы)
- Supabase (AI-чат)

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

## Сборка

```bash
npm run build
```

Статические файлы появятся в папке `dist/`.
